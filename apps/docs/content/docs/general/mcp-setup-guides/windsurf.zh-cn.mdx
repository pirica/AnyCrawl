# 在 Windsurf 中使用 MCP 网页搜索和抓取

通过 AnyCrawl MCP 为 Windsurf 添加网页抓取和搜索功能。

## 快速设置

### 1. 获取 API 密钥

在 [anycrawl.dev](https://anycrawl.dev) 注册并从控制面板复制您的 API 密钥。

### 2. 添加到 Windsurf

将以下内容添加到您的 `~/.codeium/windsurf/mcp_config.json`：

```json
{
    "mcpServers": {
        "anycrawl": {
            "command": "npx",
            "args": ["-y", "anycrawl-mcp"],
            "env": {
                "ANYCRAWL_API_KEY": "your-api-key"
            }
        }
    }
}
```

将 `your-api-key` 替换为您实际的 AnyCrawl API 密钥。

### 3. 重启 Windsurf

完成！现在 Windsurf 可以搜索和抓取网页了。

## 快速演示

在 Windsurf 中尝试以下操作：

**搜索网页：**

```
Search for the latest Tailwind CSS features
```

**抓取页面：**

```
Scrape anycrawl.dev and explain what it does
```

**获取文档：**

```
Find and scrape the Supabase authentication documentation
```

**爬取网站：**

```
Crawl the docs section of nextjs.org and summarize the key concepts
```

Windsurf 的 AI 代理将自动使用 AnyCrawl 工具。

## 故障排除

### 服务器无响应

1. 检查您的 API 密钥是否有效
2. 确保已安装 Node.js 18+
3. 更改配置后重启 Windsurf

### 权限错误

确保 npx 有执行权限。您可能需要全局安装该包：

```bash
npm install -g anycrawl-mcp
```

然后更新配置以使用全局安装的包：

```json
{
    "mcpServers": {
        "anycrawl": {
            "command": "anycrawl-mcp",
            "env": {
                "ANYCRAWL_API_KEY": "your-api-key"
            }
        }
    }
}
```
